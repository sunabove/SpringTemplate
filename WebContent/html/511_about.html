데이터 취득
아날로그 채널 입력 • 최대 64채널(전압, 전류, DC 가변가능)
    - 전압(정격 : 63.5/110V/190V/240V)
    - 측정범위
        전압 : 0.1V~150V(±0.5%), 150V~ (±1.0%)
        전류 : 0.06A~50A(±1.0%), 50.0A~(±1.0%이상)
    - 영상전압(3상 전압 입력 시 내부연산 가능)
    - 전류(정격 : 5A or 1A)
    - 입력부담 : 1VA 미만

• 특수사양
    - (DC전류 : 4 ~ 20mA)
    - (DC전압 : -10V ~ +10V)
디지털 채널 입력 • 최대 128채널(가변가능)
    - Dry접점(DC 24V)
    - Wet접점(DC48V, DC125V)
    - 입력부담 : 1VA 미만
디지털 출력 채널 • 총 4채널
    - 트리거 발생
    - 프린터 에러
    - 통신 에러
    - 시스템에러

• 릴레이 출력(A/B접점 선택)
    - 125VAC / 0.6A
    - 110VDC / 0.6A
    - 30VDC / 2.0A
데이터 샘플링 • 샘플링 주파수 : 7,680Hz(128개/Cycle)

• A/D 해상도 : 20 비트(16비트 AD+4비트)
    (Switching방식)

• 샘플링 방식 : 전채널 동시샘플링
측정대상 기동요소
측정요소 P, Q, V, f, I, δ(위상), PF
기동요소 • 각 채널 별
    - Level 트리거
    - ROC트리거

• 각 피더 별
    - P, Q 레벨트리거
    - P, Q ROC트리거
    - P, Q Swing트리거

• 각 버스 별
    - 주파수 Level 트리거
    - 주파수 ROC트리거

• Event 트리거

• 메뉴얼 트리거

• Remote 트리거

• External 트리거
저장요소 및 저장매체
저장요소 • 트리거 데이터(확장자 : *.trgx)
    - AI 64, DI 가변 채널의 매 샘플의 순시값
        (최대 사고 전 1초 , 사고 후 10초 저장)

• 페이저 데이터(확장자 : *.phax)
    - AI 64, DI 가변채널의 1회/주기 페이저 값
        (최대 사고 전 10초, 사고 후 20분 저장)

• 로깅 데이터(확장자 : *.RDI)
    - 사용자가 선택한 데이터(PQVf,채널 페이저)를 최대 32개까지 로깅.
    - 저장주기 : 최소1초, 사용자 임의의 시간으로 설정 가능
    - 저장방법
        연속저장(현재 시점에서 계속),
        1회 저장 (특정 시간 안에 1회 저장),
        반복 저장(특정 구간을 반복해서 저장)
    - 파일의 크기를 고려해 저장시간이 1일을 넘을 때 현재까지 로깅한
        데이터를 저장하고 새로운 파일 생성, 저장.
저장장치 • CF MEMORY TYPE(32 GB 이상)
    ※ 출시년도에 따라 차이가 있을 수 있음
